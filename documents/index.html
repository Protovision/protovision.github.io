socketstream

https://github.com/protovision/socketstream

A C++ library that provides std::iostream derived classes for performing TCP/IP socket I/O for network communication. Supports formatted I/O and works with standard C++ stream manipulators as the classes properly derive from the standard stream and streambuf classes. Buffering is handled internally and direct I/O is performed for large unformatted data. Supports POSIX and Windows sockets. Supports C++ move semantics and maintains basic exception safety.

Sorting Visualizer

https://protovision.github.io/sorting-visualizer

An online single page web application for visualizing sorting algorithms. It currently supports Quicksort, merge sort, heapsort, and insertion sort. Numeric View

https://protovision.github.io/numeric-view

An online single page web application for viewing and manipulating numbers in binary. Supports two's complement integer and IEEE-754 binary floating point formats of up to 64-bits. Values can be converted between decimal and binary. The binary values can be flipped, shifted, incremented, or decremented. Individual bits can be selected and flipped.
